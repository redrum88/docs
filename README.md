# docs

A public, living collection of documentation written in Markdown.

This repository is intended to grow over time and may include notes, guides, and references for different programming languages, tools, and workflows. Each topic is organized into `.md` files so it’s easy to read on GitHub and simple to contribute to.

## What you’ll find here

- Language notes and quick references (e.g., JavaScript/TypeScript, Python, Go, Rust, Java, etc.)
- Tutorials and step-by-step guides
- Snippets and examples
- Tooling and workflow docs (CLI, Git, CI/CD, editors, frameworks)

## Structure

Documentation is organized into folders by topic/language.

- Add new docs as Markdown files (`.md`)
- Keep filenames descriptive (kebab-case is preferred)
- Include code blocks and examples where helpful

## Contributing

Contributions are welcome.

- Fix typos or clarify explanations
- Add new guides or examples
- Improve organization as the repo grows

If you’re adding a new section, consider creating a folder for the language/topic and placing related `.md` files inside it.
